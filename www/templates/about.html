About Daily Loss

As the name implies, this site will have a different loss.jpg image each day. For those unfamiliar, here's a brief summary of the meme. (Congratulations, you are one of today's ten thousand!) I threw everything together over just a few days, so I don't have as many loss.jpg images as I would like. If you have any, please send them my way (even if you think I've already seen them) via this form (also linked in the header).

I've had this idea in my back pocket for a while, but I figured I should actually do something about it since New Year's is approaching. For anyone that is on the same part of the Internet as me, this was partially inspired by Alpharad Gold.

Anyways, I wanted to spread this across the Internet as much as I could, so I've also made a few bots to cross post these images on a few different social media accounts. This site is also:

  • Discord as Daily Loss Bot#5278
  • Mastodon as @dailylossbot@mastodon.social
  • Bluesky as @dailylossbot.bsky.social

Lastly, this bot is open source on GitHub. If anyone has a favorite social media platform (or other medium) that I haven't added yet, feel free to make a pull request to add it. Likewise, if you spot a bug, typo, whatever, submit the feedback form, open an issue, or use smoke signals to let me know so I can fix it.
